# 🖼️ Image Gallery

A responsive image gallery built using HTML and CSS. The project showcases a clean and organized layout designed to provide an optimal viewing experience across different screen sizes.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop devices
* Clean and modern user interface
* Organized image layout using CSS
* Smooth hover effects
* User-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Responsive Design

## 📂 Project Structure

```text
Image-Gallery/
│
├── index.html
├── style.css
└── images/
```


## 🎯 Purpose

This project was developed to improve my understanding of responsive web design, CSS layouts, and creating visually appealing user interfaces.
